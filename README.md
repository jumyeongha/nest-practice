# 과제 프로젝트

## 기능

### 투표 관리 API

- 투표 상세 조회
- 후모자 목록 조회
- 투표 목록 조회

### 투표 실행 API

- 특정 후보자에게 투표 기능

### 검색 API

- 후보자 검색

## 소프트웨어 아키텍처

- 레이어드 아키텍처 구조를 사용
  - 컨트롤러 -> 서비스 -> 레포지토리
  - 프리즈마와 결합도를 낮추기 위해 래핑하여 사용
  - 도메인 객체와 프리즈마 객체를 분리하여 도메인 로직을 순수하게 유지하도록 설계

- 크게 투표 모듈과 persistent 모듈로 설계했습니다.
  - 하나의 큰 투표 도메인의 작은 서브 도메인 느낌이라고 판단하여 모듈 분리를 하지 않았습니다.
